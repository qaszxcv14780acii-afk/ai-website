# موقع الذكاء الاصطناعي

موقع ويب عصري وحديث باللغة العربية يتحدث عن الذكاء الاصطناعي ونشأته وأفضل الأدوات المستخدمة حالياً.

## المميزات

- **ثلاث صفحات رئيسية:**
  - الصفحة الرئيسية: معلومات شاملة عن الذكاء الاصطناعي ونشأته
  - صفحة الأدوات: عرض أفضل 20 أداة ذكاء اصطناعي مع إمكانية البحث
  - صفحة التفاصيل: معلومات مفصلة عن كل أداة

- **تصميم عصري:**
  - واجهة مستخدم جذابة ومريحة للعين
  - ألوان متناسقة وجميلة
  - تصميم متجاوب يعمل على جميع الأجهزة
  - تأثيرات حركية سلسة

- **تقنيات مستخدمة:**
  - HTML5
  - Bootstrap 5
  - JavaScript (Vanilla)
  - CSS3 مع تأثيرات متقدمة

## الملفات

- `index.html` - الصفحة الرئيسية
- `tools.html` - صفحة الأدوات
- `tool-detail.html` - صفحة تفاصيل الأداة
- `styles.css` - ملف التنسيقات المخصصة
- `script.js` - ملف JavaScript الرئيسي
- `tools-data.js` - بيانات الأدوات (20 أداة)

## كيفية الاستخدام

1. افتح الملف `index.html` في المتصفح
2. تصفح الصفحات من خلال القائمة العلوية
3. في صفحة الأدوات، يمكنك البحث عن أداة معينة
4. انقر على أي أداة لعرض التفاصيل الكاملة

## الأدوات المعروضة

الموقع يعرض أفضل 20 أداة ذكاء اصطناعي في مختلف المجالات:
- أدوات النص (ChatGPT, Claude, Bard)
- أدوات الصور (Midjourney, DALL-E, Stable Diffusion)
- أدوات الفيديو (Runway ML, Synthesia)
- أدوات البرمجة (GitHub Copilot)
- أدوات الإنتاجية (Notion AI, Fireflies)
- وأدوات أخرى متنوعة
